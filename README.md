# assembler\_v2
A simple Assembler + Virtual Machine project that compiles and executes assembly code defined by a custom Instruction Set.  

# Features
* Assembly -> Machine Code Compilation
* Custom Instruction Set
* Virtual Machine for execution
* Sample programs included

# Usage
You need to compile both the compiler.baspe and vm.baspe with the latest QB64 Phoenix Edition Compiler.  
```
compiler programs/01_hello_world.asm
vm programs/01_hello_world
```

# Sample Programs
Some sample programs given are:
* Hello World
* Calculator
* Factorial
* Array Sum
* Palindrome Checker
